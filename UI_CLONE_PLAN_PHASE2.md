# DietOn 대다모(daedamo.com) UI 클론 마스터플랜 Phase 2

## 🎯 목표
기본적인 레이아웃(게시판, 상세화면, 마이페이지) 클론이 완료됨에 따라, 인터랙티브한 컴포넌트(글쓰기 폼, 사이드바 로그인, 기록 다이어리)들을 대다모의 사용자 경험에 맞춰 고도화합니다.

## 🛠 단계별 구현 계획 (Phase 2)

### 1단계: 글쓰기 에디터 폼 모달(Modal) 분리 및 개선
- [ ] 현재 `community.html` 본문에 숨겨져 펼쳐지는 글쓰기 폼을 팝업 모달 형태로 분리
- [ ] 대다모 스타일의 깔끔한 모달 헤더와 에디터 폼 영역(카테고리 선택, 제목, 내용) 디자인
- [ ] 모달 배경(오버레이) 클릭 시 닫기 및 부드러운 애니메이션 적용

### 2단계: 메인/서브 사이드바(Sidebar) 로그인 박스 고도화
- [ ] `UIManager.js`에서 렌더링하는 사이드바 로그인 영역을 대다모 스타일의 탭 기반(회원/비회원) UI로 변경
- [ ] 간편 로그인(카카오, 네이버, 구글) 버튼 디자인 통일
- [ ] 파트너 광고 배너 및 빠른 메뉴 카드형 레이아웃 개선

### 3단계: 체중 및 식단 기록(`record.html`) 다이어리 UI 클론
- [ ] 단순 인풋 폼 구조를 벗어나, 대다모의 다이어리/캘린더 스타일 입력 레이아웃으로 변경
- [ ] 아침/점심/저녁/간식 입력란 시각적 분리
- [ ] 하단 차트 위젯 카드 디자인 통일

### 4단계: 모바일 반응형(Responsive) 디테일 개선
- [ ] 모바일 환경에서 햄버거 메뉴를 눌렀을 때 좌측에서 슬라이드 인(Slide-in) 되는 네비게이션 메뉴 적용
- [ ] 하단 고정 탭바(Bottom Tab Bar) 메뉴 적용 (홈, 커뮤니티, 글쓰기, 기록, 마이)
