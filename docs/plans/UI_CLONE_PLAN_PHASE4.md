# DietOn 대다모 UI 클론 코딩 마스터플랜 - Phase 4 (서브페이지 100% 긁어오기)

기존의 눈대중으로 비슷하게 만들던 방식을 전면 폐기합니다.
사용자님의 지시("사이트를 긁어오는 수준으로 똑같이 만들어봐. 폭 높이 다 따라해")에 완벽히 부합하도록, **남은 모든 서브페이지도 대다모 원본 HTML/CSS 소스코드를 물리적으로 100% 복사해서 이식하는 초강수 미러링 작업**을 진행합니다.

## 📌 Phase 4 1:1 물리적 이식 로드맵

### 1단계: 커뮤니티 목록 페이지 (community.html) 100% 이식
- **대상**: 대다모의 '탈모수다(스토리)' 게시판 목록 페이지
- **작업**: 원본 게시판의 테이블 구조, 상단 카테고리 탭, 글쓰기 버튼 위치, 페이징(Pagination) HTML 코드를 그대로 복사하여 `community.html`의 `<main>` 영역을 통째로 갈아 끼웁니다.
- **수정**: '탈모'를 '다이어트'로 텍스트만 치환.

### 2단계: 게시글 상세 페이지 (post.html) 100% 이식
- **대상**: 대다모의 개별 게시글 읽기 페이지
- **작업**: 본문 제목 폰트 크기, 작성자 아바타 이미지 영역, 본문 레이아웃, 하단의 '추천/비추천' 버튼 디자인, 그리고 댓글(Comment) 영역의 마크업 소스를 원본에서 가져와 덮어씌웁니다.

### 3단계: 마이페이지 (my.html) 100% 이식
- **대상**: 대다모 마이페이지 (내 정보, 작성 글 확인)
- **작업**: 좌측의 좁은 LNB(로컬 내비게이션 바)와 우측의 내 정보 확인 콘텐츠 테이블의 원본 HTML 구조를 복사하여 적용합니다.

### 4단계: 다이어트 기록 (record.html) 100% 이식
- **대상**: 대다모 사진 업로드/글쓰기 폼
- **작업**: 대다모의 에디터 형태와 사진 첨부 영역 디자인을 그대로 긁어와서 다이어트 식단 및 체중 인증 폼으로 변경합니다.

---
**진행 방식**: 더 이상의 "비슷하게 흉내내기"는 없습니다. 폭, 높이, 폰트, 여백 등 모든 것을 대다모의 **원본 코드 복붙**으로 해결하여 100% 완벽한 샴쌍둥이를 만들겠습니다.
