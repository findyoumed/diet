# DietOn 대다모(dieton.com) UI 클론 마스터플랜

## 🎯 목표
DietOn 커뮤니티를 대다모 웹사이트의 검증된 레이아웃, 컬러체계, UX 패턴과 동일하게 개편합니다.

## 🛠 단계별 구현 계획

### 1단계: 메인 페이지(Home) 고도화 및 GNB (완료/진행중)
- [x] 메인 색상 보라색 테마 적용 (`style.css`)
- [x] 상단 퀵메뉴 및 알림바 구성 (`index.html`)
- [x] 포토 갤러리 및 Q&A 카드형 레이아웃 (`app.js`)
- [ ] GNB(Global Navigation Bar) 2뎁스 드롭다운 메뉴 적용
- [ ] 모바일 반응형 하단 플로팅 네비게이션 바 추가

### 2단계: 게시판 목록 페이지 (`community.html`) UI 클론
- [ ] 대다모 스타일의 밀도 높은 테이블형 게시글 리스트 레이아웃 (번호, 분류, 제목, 닉네임, 날짜, 조회수)
- [ ] 게시물 제목 옆에 댓글 수 `[12]`, 첨부파일 아이콘, N(New) 뱃지 표시
- [ ] 상단 게시판 헤더 및 카테고리(전체/식단/비포애프터 등) 탭 디자인 통일
- [ ] 하단 페이징(Pagination) 및 검색창 컨트롤 디자인

### 3단계: 게시글 상세 페이지 (`post.html`) UI 클론
- [ ] 게시물 제목, 작성자, 작성일, 조회수 등 상단 헤더 박스 디자인
- [ ] 하단 중앙의 눈에 띄는 '추천/비추천' 버튼 디자인
- [ ] 대다모 스타일의 댓글 리스트 (프로필, 닉네임, 시간, 내용) 및 댓글 입력 폼 디자인

### 4단계: 회원/마이페이지 (`my.html`)
- [ ] 좌측 사이드바(LNB) 메뉴와 우측 콘텐츠 2단 레이아웃
- [ ] 내 정보 수정, 포인트 내역 등 폼 및 테이블 UI 개선
